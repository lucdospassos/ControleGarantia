Controle de Garantias

Seus dados ficam salvos neste celular, sem internet.

Backup automático em arquivo
Vincule um arquivo .json uma vez. A partir daí, todo registro salvo aqui também é gravado automaticamente nesse arquivo.
Backup automático indisponível
Seu navegador ainda não permite gravar direto num arquivo do celular. Os dados continuam salvos aqui normalmente — use "Exportar backup" com frequência para ter uma cópia de segurança em .json.
Backup automático (qualquer navegador)
Ativando isso, toda vez que você salvar um registro o app baixa sozinho uma cópia completa e atualizada em .json para a pasta Downloads — funciona mesmo sem suporte a "vincular arquivo". Como o navegador não sobrescreve arquivos, as cópias vão se acumular ali como "garantias-auto-backup (1).json", "(2)"... de vez em quando vale apagar as mais antigas.
