GERAR APK PELO ANDROID STUDIO

1. Abra o Android Studio.
2. File > Open.
3. Selecione a pasta ControleLacresGarantia.
4. Aguarde sincronizar o Gradle.
5. Para teste:
   Build > Build Bundle(s) / APK(s) > Build APK(s)
6. O APK debug ficará em:
   app/build/outputs/apk/debug/app-debug.apk

Para APK release assinado pelo Android Studio:
1. Build > Generate Signed Bundle / APK.
2. Escolha APK.
3. Use ou crie uma chave .jks.
4. Gere a versão release.

Sem Android Studio, use o GitHub Actions conforme GERAR_APK_GITHUB_ACTIONS.txt.
