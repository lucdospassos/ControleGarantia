GERAR APK LOCALMENTE

Opção com Android Studio:
1. Abra a pasta ControleLacresGarantia no Android Studio.
2. Aguarde sincronizar.
3. Build > Build Bundle(s) / APK(s) > Build APK(s).

Opção recomendada sem Android Studio:
Use o GitHub Actions conforme GERAR_APK_GITHUB_ACTIONS.txt.

Versão do projeto: 1.5.0 estável.
