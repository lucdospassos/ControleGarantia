GERAR APK PELO GITHUB ACTIONS

1. Envie/substitua os arquivos deste projeto no repositório GitHub.
2. Confira se existe:

   .github/workflows/build-apk.yml

3. Vá em:

   Actions > Gerar APK Android > Run workflow

4. Se não houver secrets de assinatura, será gerado apenas o APK debug:

   ControleLacresGarantia-v1.5-debug-apk

5. Se os 4 secrets de assinatura estiverem configurados, também será gerado:

   ControleLacresGarantia-v1.5-release-assinado-apk

Para configurar a versão release assinada, leia:
CONFIGURAR_RELEASE_ASSINADO_GITHUB_ACTIONS.txt
