GERAR APK PELO GITHUB ACTIONS

1. Envie todos os arquivos desta pasta para o GitHub, mantendo a estrutura:

.github/workflows/build-apk.yml
app/
docs/
web/
build.gradle
settings.gradle
README.txt

2. No GitHub, abra:
Actions > Gerar APK Android

3. Clique em:
Run workflow

4. Quando terminar, baixe o artifact:
ControleLacresGarantia-debug-apk

Dentro estará o APK debug, bom para teste.

5. Para gerar o APK release assinado, configure os Secrets descritos em:
CONFIGURAR_RELEASE_ASSINADO_GITHUB_ACTIONS.txt

Depois rode o mesmo workflow novamente.
Quando a assinatura estiver configurada, o GitHub também vai gerar o artifact:
ControleLacresGarantia-release-assinado-apk

Observação:
- O APK debug é para testes.
- O APK release assinado é o recomendado para entregar aos clientes.
